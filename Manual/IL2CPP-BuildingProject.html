    Building a project using IL2CPP

    To build your project using IL2CPP, open the Build Settings window (File > Build Settings). Select the platform you are building for, then click Player SettingsSettings that let you set various player-specific options for the final game built by Unity. More info

    See in Glossary    .

    In the Project SettingsA broad collection of settings which allow you to configure how Physics, Audio, Networking, Graphics, Input and many other areas of your Project behave. More info
    See in Glossary     window, locate the Player settings for your target platform, and scroll down to the Configuration section. For Scripting BackendA framework that powers scripting in Unity.
