    2D Tilemap Editor

    com.unity.2d.tilemap

    Description

    2D Tilemap Editor is a package that contains editor functionalities for editing Tilemaps.

    Version information

    Compatible with Unity

    These package versions are compatible with Unity version 2019.2:

    Documentation location: Versions supported:
    com.unity.2d.tilemap@1.0 1.0.0

    Verified for Unity

    Package version 1.0.0 is verified to work with 2019.2.
