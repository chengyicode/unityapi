LightmapEditorSettings.filteringGaussRadiusAO

public static int filteringGaussRadiusAO;

Description 描述

Specifies the radius the Progressive lightmapper uses to filter for ambient occlusion in the lightmap when using Gaussian filter. The default value is 2. The value range is 0 to 5.

A higher value will increase the strength of the blur, reducing noise for ambient occlusion in the lightmap.
