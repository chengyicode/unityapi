Description 描述

Determines the orientation of tiles in the Tilemap.

确定Tilemap中瓦片的方向。

Properties 属性

xy XY 在XY平面中确定瓦片朝向。 Orients tiles in the XY plane.
xz XZ 在XZ平面中确定瓦片朝向。 Orients tiles in the XZ plane.
yx YX 在YX平面中确定瓦片朝向。 Orients tiles in the YX plane.
yz YZ 在YZ平面中确定瓦片朝向。 Orients tiles in the YZ plane.
zx ZX 在ZX平面中确定瓦片朝向。 Orients tiles in the ZX plane.
zy ZY 在ZY平面中确定瓦片朝向。 Orients tiles in the ZY plane.
自定义 Custom 对瓦片地图中的所有瓦片使用自定义方向。 Use a custom orientation to all tiles in the tile map.
