AtomicSafetyHandle.GetReaderArray

public static int GetReaderArray(Unity.Collections.LowLevel.Unsafe.AtomicSafetyHandle handle, int maxCount, IntPtr output);

Parameters

handleThe atomic safety handle to return readers for.
maxCountThe maximum number of handles to be written to the output array.
outputA buffer where the job handles will be written.

Returns

int The actual number of readers on the handle, which can be greater than the maximum count provided.

Description 描述

Fetch the job handles of all jobs reading from the safety handle.

This is a debugging method used to produce better error messages and is not intended to be called from user code.
