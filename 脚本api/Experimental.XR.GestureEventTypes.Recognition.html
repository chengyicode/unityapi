GestureEventTypes.Recognition

Description 描述

Flag indicating a recognition gesture.

The platform tracks the position of hands that are visible to the device in order to recognize hand gestures.The platform sees hands when they are in the ready state (back of the hand facing you with index finger up), or the pressed state (back of the hand facing you with the index finger down). The platform ignores hands if they are in other poses.
