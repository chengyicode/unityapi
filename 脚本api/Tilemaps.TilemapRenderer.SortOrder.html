SortOrder 排序规则

enumeration 枚举

Description 描述

Sort order for all tiles rendered by the TilemapRenderer.

对TilemapRenderer渲染的所有图块的排序顺序。

Properties 属性

左下 BottomLeft 从具有最低X和最低Y单元格位置的图块开始，对要渲染的图块进行排序。 Sorts tiles for rendering starting from the tile with the lowest X and the lowest Y cell positions.
右下 BottomRight 从具有最高X单元格位置和最低Y单元格位置的图块开始，对要渲染的图块进行排序。 Sorts tiles for rendering starting from the tile with the highest X and the lowest Y cell positions.
左上 TopLeft 从具有最低X单元格位置和最高Y单元格位置的图块开始，对要渲染的图块进行排序。 Sorts tiles for rendering starting from the tile with the lowest X and the highest Y cell positions.
