PrefabUtility.IsPartOfPrefabThatCanBeAppliedTo

public static bool IsPartOfPrefabThatCanBeAppliedTo(Object gameObjectOrComponent);

Parameters

gameObjectOrComponentThe object to check. Must be a component or GameObject.

Returns

bool True if the object is part of a Prefab that cannot be applied to.

Description 描述

Is this object part of a Prefab that cannot be applied to?

For an object on a Prefab Asset, the asset itself is checked. For an object on a Prefab instance, its corresponding asset is checked.

Examples of Prefab Assets that cannot be applied to are immutable Prefab Assets, and Prefab Assets with invalid components where the script is missing.
