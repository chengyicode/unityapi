Editor.CreateCachedEditorWithContext

public static void CreateCachedEditorWithContext(Object targetObject, Object context, Type editorType, ref Editor previousEditor);
public static void CreateCachedEditorWithContext(Object[] targetObjects, Object context, Type editorType, ref Editor previousEditor);

Description 描述

Creates a cached editor using a context object.

This method is similar to CreateCachedEditor except that the SerializedObject is created with the context object passed as a parameter. This context object stores and retrieves the value for the ExposedReference types declared on the SerializedObject.
