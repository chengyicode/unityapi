Mode 模式

enumeration 枚举

Description 描述

Determines how the TilemapRenderer should batch the sprites from tiles for rendering.

确定TilemapRenderer如何批处理来自图块的精灵以进行渲染。

Properties 属性

区块 Chunk 将来自Tilemap中的每个Sprite批处理为分组的区块以进行渲染。 Batches each Sprite from the Tilemap into grouped chunks to be rendered.
独立(个体) Individual 将来自Tilemap的每个Sprite进行单独渲染。 Sends each Sprite from the Tilemap to be rendered individually.
