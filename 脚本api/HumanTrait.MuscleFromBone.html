HumanTrait.MuscleFromBone

public static int MuscleFromBone(int i, int dofIndex);

Parameters

iBone index.
dofIndexNumber representing a "degree of freedom": 0 for X-Axis, 1 for Y-Axis, 2 for Z-Axis.

Description 描述

Obtain the muscle index for a particular bone index and "degree of freedom".

The indexing order of the bones is the same as that of the BoneName array.

See Also: HumanTrait.BoneName, HumanTrait.BoneCount, HumanTrait.MuscleName, HumanTrait.MuscleCount.
