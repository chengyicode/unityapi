AudioImporter.GetOverrideSampleSettings

public AudioImporterSampleSettings GetOverrideSampleSettings(string platform);

Parameters

platformThe platform to get the override settings for.

Returns

AudioImporterSampleSettings The override sample settings for the given platform.

Description 描述

Return the current override settings for the given platform.

If there is no override for the passed platform, a defaults translated settings will be returned.

The translated settings are what settings will be used by default for the given platform. For example, on some platforms, the compressionFormat will be different to capitalise on hardware decoders.

The options for the platform string are "WebPlayer", "Standalone", "iOS", "Android", "WebGL", "PS4", "XBoxOne".
