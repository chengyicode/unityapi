Mesh.GetBonesPerVertex

Switch to Manual
public NativeArray<byte> GetBonesPerVertex();

Description 描述

The number of non-zero bone weights for each vertex.

The size of the returned array is either vertexCount or zero.

Use in combination with GetAllBoneWeights to get bone weights for the vertices in the Mesh. Unlike boneWeights, these methods are not limited to 4 bones per vertex.

See Also: SetBoneWeights.
