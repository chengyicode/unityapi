Physics2D.SetLayerCollisionMask

public static void SetLayerCollisionMask(int layer, int layerMask);

Parameters

layerThe layer to set the collision layer mask for.
layerMaskA mask where each bit indicates a layer and whether it can collide with layer or not.

Description 描述

Set the collision layer mask that indicates which layer(s) the specified layer can collide with.

See Also: GetLayerCollisionMask, IgnoreLayerCollision.
