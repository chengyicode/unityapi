HierarchyFrameDataView.GetItemMergedSamplesCount

public int GetItemMergedSamplesCount(int id);

Parameters

idHierarchy item identifier.

Returns

int Merged samples count represented by the tree item.

Description 描述

Return merged samples count represented by the hierarchy item.

When HierarchyFrameDataView uses ViewMode.MergeSamplesWithTheSameName mode, multiple samples with the same name at the same hierarchy level are merged together and are represented by a single item.
