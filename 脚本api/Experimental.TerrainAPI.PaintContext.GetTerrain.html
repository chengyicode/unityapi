PaintContext.GetTerrain

public Terrain GetTerrain(int terrainIndex);

Parameters

terrainIndexIndex of the terrain.

Returns

Terrain Returns the Terrain object.

Description 描述

Retrieves a Terrain from the PaintContext.

When painting across a border, the PaintContext can refer to several Terrains. GetTerrain is used to access those Terrains. terrainIndex must be between 0 and PaintContext.terrainCount - 1. See Also: PaintContext.GetClippedPixelRectInTerrainPixels and PaintContext.GetClippedPixelRectInRenderTexturePixels.
