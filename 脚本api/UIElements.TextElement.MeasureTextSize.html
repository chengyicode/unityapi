TextElement.MeasureTextSize

public Vector2 MeasureTextSize(string textToMeasure, float width, UIElements.VisualElement.MeasureMode widthMode, float height, UIElements.VisualElement.MeasureMode heightMode);

Parameters

textToMeasureThe text to measure.
widthSuggested width. Can be zero.
widthModeWidth restrictions.
heightSuggested height.
heightModeHeight restrictions.

Returns
