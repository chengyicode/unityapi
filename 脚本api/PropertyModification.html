Description 描述

Defines a single modified property.

Used by the Prefab system to track any changes applied to an instance.

Properties 属性

objectReferenceThe value being applied when it is a object reference (which can not be represented as a string).
propertyPathProperty path of the property being modified (Matches as SerializedProperty.propertyPath).
targetObject that will be modified.
valueThe value being applied.
