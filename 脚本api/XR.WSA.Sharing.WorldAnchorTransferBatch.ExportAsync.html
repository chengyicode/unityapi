WorldAnchorTransferBatch.ExportAsync

public static void ExportAsync(XR.WSA.Sharing.WorldAnchorTransferBatch transferBatch, XR.WSA.Sharing.WorldAnchorTransferBatch.SerializationDataAvailableDelegate onDataAvailable, XR.WSA.Sharing.WorldAnchorTransferBatch.SerializationCompleteDelegate onCompleted);

Parameters

transferBatchThe WorldAnchorTransferBatch to export into a byte buffer.
onDataAvailableThe callback when some data is available.
onCompletedThe callback after the last bit of data was provided via onDataAvailable with a completion reason (success or failure reason).

Description 描述

Exports the input WorldAnchorTransferBatch into a byte array which can be passed to WorldAnchorTransferBatch.ImportAsync to restore the original WorldAnchorTransferBatch.
