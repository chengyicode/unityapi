RefreshMode

enumeration

Switch to Manual

Description 描述

An enum describing the way a Light Probe Proxy Volume refreshes in the Player.

See Also: Light Probe Proxy Volume.

Properties 属性

AutomaticAutomatically detects updates in Light Probes and triggers an update of the Light Probe volume.
EveryFrameCauses Unity to update the Light Probe Proxy Volume every frame.
ViaScriptingUse this option to indicate that the Light Probe Proxy Volume is never to be automatically updated by Unity.
