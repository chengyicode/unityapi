ParticleSystem.ShapeModule.radiusSpeedMultiplier

Switch to Manual
public float radiusSpeedMultiplier;

Description 描述

A multiplier of the radius speed of the particle emission shape.

This method is more efficient than accessing the whole curve, if you only want to change the overall speed multiplier. See Also: ParticleSystem.ShapeModule.radiusMode, ParticleSystemShapeMultiModeValue, ParticleSystem.ShapeModule.shapeType.
