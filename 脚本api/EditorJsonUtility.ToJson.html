EditorJsonUtility.ToJson

public static string ToJson(object obj);
public static string ToJson(object obj, bool prettyPrint);

Parameters

objThe object to convert to JSON form.
prettyPrintIf true, format the output for readability. If false, format the output for minimum size. Default is false.

Returns

string The object's data in JSON format.

Description 描述

Generate a JSON representation of an object.

This is similar to JsonUtility.ToJson, but it supports any engine object. The fields that the output will contain are the same as are accessible via the SerializedObject API, or as found in the YAML-serialized form of the object.
