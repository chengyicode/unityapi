GestureEventState

enumeration

Description 描述

The state of a given gesture. A gesture can be thought of as a state machine that transitions through multiple states during it's lifetime.

Gestures are Started, then Updated while active, then either Completed or Canceled. Some gestures however don't have a continuous lifetime, and instead have a single Discrete state.

Properties 属性

DiscreteThis state describes when a gesture with a single discrete event occurs. There will be no more events triggered for this gesture.
StartedThis state describes when a gesture has been started. The gesture may be updated before completing or being cancelled.
UpdatedThis state describes when a gesture is being updated during it's lifetime.
CompletedThis state describes when the user completes the gesture. Unity considers this gesture to be finished and no longer receives data for it.
CanceledThis state describes when the user cancels the gesture. Unity considers this gesture to be finished and no longer receives data for it.
