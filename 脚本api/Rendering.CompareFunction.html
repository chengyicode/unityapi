CompareFunction

enumeration

Description 描述

Depth or stencil comparison function.

Properties 属性

DisabledDepth or stencil test is disabled.
NeverNever pass depth or stencil test.
LessPass depth or stencil test when new value is less than old one.
EqualPass depth or stencil test when values are equal.
LessEqualPass depth or stencil test when new value is less or equal than old one.
GreaterPass depth or stencil test when new value is greater than old one.
NotEqualPass depth or stencil test when values are different.
GreaterEqualPass depth or stencil test when new value is greater or equal than old one.
AlwaysAlways pass depth or stencil test.
