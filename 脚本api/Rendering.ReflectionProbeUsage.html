ReflectionProbeUsage

enumeration

Description 描述

Reflection Probe usage.

Properties 属性

OffReflection probes are disabled, skybox will be used for reflection.
BlendProbesReflection probes are enabled. Blending occurs only between probes, useful in indoor environments. The renderer will use default reflection if there are no reflection probes nearby, but no blending between default reflection and probe will occur.
BlendProbesAndSkyboxReflection probes are enabled. Blending occurs between probes or probes and default reflection, useful for outdoor environments.
SimpleReflection probes are enabled, but no blending will occur between probes when there are two overlapping volumes.
