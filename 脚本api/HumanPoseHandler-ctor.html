HumanPoseHandler Constructor

public HumanPoseHandler(Avatar avatar, Transform root);

Parameters

avatarThe avatar that defines the humanoid rig on skeleton hierarchy with root as the top most parent.
rootThe top most node of the skeleton hierarchy defined in humanoid avatar.

Description 描述

Creates a human pose handler from an avatar and a root transform.

The avatar must be humanoid and skeleton hierarchy root must match the avatar definition.
