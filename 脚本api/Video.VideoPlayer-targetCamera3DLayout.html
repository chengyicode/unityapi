VideoPlayer.targetCamera3DLayout

public Video.Video3DLayout targetCamera3DLayout;

Description 描述

Type of 3D content contained in the source video media.

When PlayerSettings.virtualRealitySupported is enabled, rendering for each eye will sample the correct half of the video according to this setting. Only used when VideoPlayer.renderMode is set to either Video.VideoTarget.CameraBackPlane or Video.VideoTarget.CameraFrontPlane.
