BuiltinPaintMaterialPasses

enumeration

Description 描述

Enumeration of the render passes in the built-in paint material.

Properties 属性

RaiseLowerHeightBuilt-in render pass for raising and lowering terrain height.
StampHeightBuilt-in render pass for stamping heights on the terrain.
SetHeightsBuilt-in render pass for setting terrain height.
SmoothHeightsBuilt-in render pass for smoothing the terrain height.
PaintTextureBuilt-in render pass for painting the splatmap texture.
