ContactFilter2D.NoFilter

public ContactFilter2D NoFilter();

Returns

ContactFilter2D A copy of the contact filter set to not filter any ContactPoint2D.

Description 描述

Sets the contact filter to not filter any ContactPoint2D.

When you are using a function that requires a ContactFilter2D, but you don't want to perform any filtering, then use this function to return a ContactFilter2D that turns off any filtering.

Call this function to set the contact filter to layerMask = Physics2D.AllLayers, minDepth = Mathf.Infinity, maxDepth = Mathf.Infinity, minNormalAngle = Mathf.Infinity and maxNormalAngle = Mathf.Infinity.
