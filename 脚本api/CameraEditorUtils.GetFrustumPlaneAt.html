CameraEditorUtils.GetFrustumPlaneAt

public static void GetFrustumPlaneAt(Matrix4x4 clipToWorld, Vector3 viewPosition, float distance, Vector3[] points);

Parameters

clipToWorldClip space to world space matrix.
viewPositionView position in world space.
distanceDistance from the view position of the plane.
pointsCalculated points. (A minimum size of 4 elements is required).

Description 描述

Calculate the points of the frustrum plane facing the viewer at a specific distance.

The points array will be filled with the calculated points in the following order: left bottom, left top, right top and right bottom.
