WorldUpType

enumeration

Description 描述

Specifies how the world up vector used by the aim constraint is defined.

Properties 属性

SceneUpUses and defines the world up vector as the Unity Scene up vector (the Y axis).
ObjectUpUses and defines the world up vector as a vector from the constrained object, in the direction of the up object.
ObjectRotationUpUses and defines the world up vector as relative to the local space of the object.
VectorUses and defines the world up vector as a vector specified by the user.
NoneNeither defines nor uses a world up vector.
