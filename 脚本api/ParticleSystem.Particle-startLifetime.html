ParticleSystem.Particle.startLifetime

Switch to Manual
public float startLifetime;

Description 描述

The starting lifetime of the particle.

This is the total lifetime of this particle in seconds. It is set by the Particle System when creating the particle.

using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        ParticleSystem.Particle particle = new ParticleSystem.Particle();

        // Calculate how long the particle has been alive.
        float timeAlive = particle.startLifetime - particle.lifetime;
    }
}
