UnityEngine.UnityWebRequestAudioModule

Description 描述

The UnityWebRequestAudio module provides the DownloadHandlerAudioClip class to use UnityWebRequest to download AudioClips.

Classes

DownloadHandlerAudioClipA DownloadHandler subclass specialized for downloading audio data for use as AudioClip objects.
UnityWebRequestMultimediaHelpers for downloading multimedia files using UnityWebRequest.
