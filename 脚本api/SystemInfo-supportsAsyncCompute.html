SystemInfo.supportsAsyncCompute

public static bool supportsAsyncCompute;

Description 描述

Returns true when the platform supports asynchronous compute queues and false if otherwise.

Note that asynchronous compute queues are only supported on PS4.

See Also: ComputeShader, ComputeBuffer, CommandBuffer classes. ScriptableRenderContext.ExecuteCommandBufferAsync, Graphics.ExecuteCommandBufferAsync methods.
