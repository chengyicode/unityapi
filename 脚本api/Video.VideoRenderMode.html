VideoRenderMode

enumeration

Description 描述

Type of destination for the images read by a VideoPlayer.

Properties 属性

CameraFarPlaneDraw video content behind a camera's Scene.
CameraNearPlaneDraw video content in front of a camera's Scene.
RenderTextureDraw video content into a RenderTexture.
MaterialOverrideDraw the video content into a user-specified property of the current GameObject's material.
APIOnlyDon't draw the video content anywhere, but still make it available via the VideoPlayer's texture property in the API.
