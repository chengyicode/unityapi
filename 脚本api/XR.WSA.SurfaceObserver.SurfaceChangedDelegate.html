SurfaceObserver.SurfaceChangedDelegate

public delegate void SurfaceChangedDelegate(XR.WSA.SurfaceId surfaceId, XR.WSA.SurfaceChange changeType, Bounds bounds, DateTime updateTime);

Parameters

surfaceIdThe ID of the surface that has changed.
changeTypeThe type of change this event represents (Added, Updated, Removed).
boundsThe bounds of the surface as reported by the device.
updateTimeThe update time of the surface as reported by the device.

Description 描述

The SurfaceChanged delegate handles SurfaceChanged events as generated by calling Update on a SurfaceObserver. Applications can use the bounds, changeType, and updateTime to selectively generate mesh data for the set of known surfaces.
