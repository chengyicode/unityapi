PlayableExtensions.GetInputWeight

public static float GetInputWeight(U playable, int inputIndex);

Parameters

playableThe Playable used by this operation.
inputIndexThe port index.

Returns

float The current weight of the connected Playable.

Description 描述

Returns the weight of the Playable connected at the given input port index.

NOTE: This is a templated extension method that can be used on any type that inherits from IPlayable
