AnimatorController.AddMotion

public Animations.AnimatorState AddMotion(Motion motion);
public Animations.AnimatorState AddMotion(Motion motion, int layerIndex);

Parameters

motionThe Motion that will be in the AnimatorState.
layerIndexThe layer where the Motion will be added.

Description 描述

Utility function that creates a new state with the motion in it.

The state asset that is created is added as a sub asset of the state machine to which it is added. This function pushes an Undo operation.
