Obsolete public Networking.ReactorModel ReactorModel;

Description 描述

Defines reactor model for the network library.

With the select model, received packets will be handled as soon as they arrive. With the fixed rate model, incoming packets will only be handled periodically. The select model is good for non-mobile devices, where the battery life is not important. Fixed rate is not as CPU intensive and will save battery life on mobile devices but this decreases performance.
