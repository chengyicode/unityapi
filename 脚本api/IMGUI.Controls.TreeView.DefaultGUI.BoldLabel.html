TreeView.DefaultGUI.BoldLabel

public static void BoldLabel(Rect rect, string label, bool selected, bool focused);

Parameters

rectRect to render the text in.
labelLabel to render.
selectedSelected state used for determining text color.
focusedFocused state used for determining text color.

Description 描述

Draws a bold label that have correct text color when selected and/or focused.

This method uses the normal, focused, onNormal and onFocused GUIStyleStates of TreeView.DefaultStyles.boldLabel to ensure correct rendering of text above a selection background.
