AnimatorController.CreateAnimatorControllerAtPathWithClip

public static Animations.AnimatorController CreateAnimatorControllerAtPathWithClip(string path, AnimationClip clip);

Parameters

pathThe path where the AnimatorController will be created.
clipThe default clip that will be played by the AnimatorController.

Description 描述

Creates an AnimatorController at the given path, and automatically create an AnimatorLayer with an AnimatorStateMachine that will add a State with the AnimationClip in it.

This function pushes an Undo operation.
