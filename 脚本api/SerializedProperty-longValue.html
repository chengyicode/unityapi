SerializedProperty.longValue

public long longValue;

Description 描述

Value of a integer property as a long.

Contains a valid value when propertyType is SerializedPropertyType.Integer. For accessing sbyte, byte, short, ushort, int, uint, long and ulong properties.

When assigning a value to intValue, the value is clamped in the range of the property's declared type. For example, a property that is declared as a byte is clamped between 0 and 255.

See Also: intValue, propertyType, SerializedPropertyType.Integer.
