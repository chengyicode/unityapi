MouseCaptureEventBase<T0>.GetPooled

public static T GetPooled(UIElements.IEventHandler target, UIElements.IEventHandler relatedTarget);

Parameters

targetThe handler taking or releasing the mouse capture.
relatedTargetThe related target.

Returns

T Returns an initialized event.

Description 描述

Retrieves an event from the event pool. Use this method to retrieve a mouse event and initialize the event, instead of creating a new mouse event. Events obtained from this method should be released back to the pool using Dispose().
