ParticleSystem.SubEmittersModule.death0

Switch to Manual
Obsolete public ParticleSystem death0;

Description 描述

Sub Particle System which spawns at the locations of the death of the particles from the parent system.

The sub-Particle System will be emitted using the first emission burst parameters. If the system does not contain burst parameters, then nothing will be emitted; if the system contains more than one burst parameters, then the additional parameters will not be used.
