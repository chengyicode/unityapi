WorldAnchor.OnTrackingChanged

Parameters

valueEvent that fires when this object's tracking state changes.

Description 描述

OnTrackingChanged notifies listeners when this object's tracking state changes.

By default, an object that loses its ability to be tracked will be deactivated. This behavior can be overridden by listening for this event. If this event is subscribed to, it becomes the developer's responsibility to handle the tracking loss state for this object.
