XRGestureSubsystem.ManipulationChanged

Parameters

valueEvent data for manipulation gesture.

Description 描述

Event that fires when the state of a manipulation gesture changes.

Manipulation gestures can be used to move, resize or rotate an object when it should react 1:1 to the user's hand movements. One use for such 1:1 movements is to let the user draw or paint in the world. The initial targeting for a manipulation gesture should be done by gaze or pointing. Once the tap and hold starts, any manipulation of the object is then handled by hand movements, freeing the user to look around while they manipulate.
