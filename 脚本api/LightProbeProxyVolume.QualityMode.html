QualityMode

enumeration

Switch to Manual

Description 描述

An enum describing the Quality option used by the Light Probe Proxy Volume component.

See Also: Light Probes, Light Probe Proxy Volume, Spherical Harmonics(SH).

Properties 属性

LowThis option will use only two SH coefficients bands: L0 and L1. The coefficients are sampled from the Light Probe Proxy Volume 3D Texture. Using this option might increase the draw call batch sizes by not having to change the L2 coefficients per Renderer.
NormalThis option will use L0 and L1 SH coefficients from the Light Probe Proxy Volume 3D Texture. The L2 coefficients are constant per Renderer. By having to provide the L2 coefficients, draw call batches might be broken.
