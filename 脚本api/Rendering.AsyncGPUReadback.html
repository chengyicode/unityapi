AsyncGPUReadback

class in UnityEngine.Rendering

/

Implemented in:UnityEngine.CoreModule

Description 描述

Allows the asynchronous read back of GPU resources.

This class is used to copy resource data from the GPU to the CPU without any stall (GPU or CPU), but adds a few frames of latency. See Also: AsyncGPUReadbackRequest.

Static Methods 静态方法

RequestTriggers a request to asynchronously fetch the data from a GPU resource.
