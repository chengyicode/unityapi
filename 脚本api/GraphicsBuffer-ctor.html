GraphicsBuffer Constructor

public GraphicsBuffer(GraphicsBuffer.Target target, int count, int stride);

Parameters

targetSelect whether this buffer can be used as a vertex or index buffer.
countNumber of elements in the buffer.
strideSize of one element in the buffer. For index buffers, this must be either 2 or 4 bytes.

Description 描述

Create a Graphics Buffer.

Use Release to release the buffer when no longer needed.
