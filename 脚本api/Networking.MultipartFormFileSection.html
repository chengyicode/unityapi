MultipartFormFileSection

Implements interfaces:IMultipartFormSection

Description 描述

A helper object for adding file uploads to multipart forms via the [IMultipartFormSection] API.

This object is very similar to the [MultipartFormDataSection] object, but all constructors additionally accept (and require) a fileName parameter. If you omit the fileName parameter, this object will provide a default filename.

Properties 属性

contentTypeReturns the value of the section's Content-Type header.
fileNameReturns a string denoting the desired filename of this section on the destination server.
sectionDataReturns the raw binary data contained in this section. Will not return null or a zero-length array.
sectionNameReturns the name of this section, if any.

Constructors

MultipartFormFileSectionContains a named file section based on the raw bytes from data, with a custom Content-Type and file name.
