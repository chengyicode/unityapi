LightmapEditorSettings.filteringGaussRadiusIndirect

public static int filteringGaussRadiusIndirect;

Description 描述

Specifies the radius the Progressive lightmapper used to filter for indirect light stored in the lightmap when using Gaussian filter. The default value is 5. The value range is 0 to 5.

A higher value will increase the strength of the blur, reducing noise from indirect light in the lightmap.
