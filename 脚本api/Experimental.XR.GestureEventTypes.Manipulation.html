GestureEventTypes.Manipulation

Description 描述

Flag indicating a manipulation gesture.

You can use Manipulation gestures to move, resize, or rotate an object when you want it to react 1:1 to hand movement. An example use for a 1:1 movement like this is to draw or paint into the Scene. You should use a gaze or point for the initial targeting of a manipulation gesture and then, when the tap and hold starts, the hand movements handle any manipulation of the object, allowing you to look around while you manipulate the object.
