ConnectionTarget

enumeration

Description 描述

The type of the connected target.

This is used in IConnectionState, for example, to signify whether the Editor is connected to a Player or to an Editor. A connection to an Editor also picks up the PlayMode Player. When an Editor is not connected to a Player, it will always fall back onto a connection to itself. ConnectionTarget.None would therefore only be applicable in a Player.

Properties 属性

NoneNo target is connected, this is only possible in a Player.
PlayerThe connected target is a Player.
EditorThe connected target is an Editor.
