RemoteSettings.GetLong

public static long GetLong(string key, long defaultValue = 0);

Parameters

keyThe key identifying the setting.
defaultValueThe default value to use if the setting identified by the key parameter cannot be found or is unavailable.

Returns

long The current value of the setting identified by key, or the default value.

Description 描述

Gets the value corresponding to remote setting identified by key, if it exists.

If it doesn't exist, it will return defaultValue. If you don't provide a default value, it will return 0.
