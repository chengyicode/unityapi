Client.ResetToEditorDefaults

public static PackageManager.Requests.ResetToEditorDefaultsRequest ResetToEditorDefaults();

Returns

ResetToEditorDefaultsRequest A ResetToEditorDefaultsRequest instance, which you can use to monitor the status of the asynchronous operation.

Description 描述

Resets the list of packages installed for this project to the editor's default configuration. This operation will clear all packages added to the project and keep only the packages set for the current editor default configuration.
