ConnectionAcksType

enumeration

Description 描述

Defines size of the buffer holding reliable messages, before they will be acknowledged.

Properties 属性

Acks32Ack buffer can hold 32 messages.
Acks64Ack buffer can hold 64 messages.
Acks96Ack buffer can hold 96 messages.
Acks128Ack buffer can hold 128 messages.
