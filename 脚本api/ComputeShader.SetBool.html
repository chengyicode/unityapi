ComputeShader.SetBool

Switch to Manual
public void SetBool(string name, bool val);
public void SetBool(int nameID, bool val);

Parameters

nameVariable name in shader code.
nameIDProperty name ID, use Shader.PropertyToID to get it.
valValue to set.

Description 描述

Set a bool parameter.

Constant buffers are shared between all kernels in a single compute shader asset. Therefore this function affects all kernels in this ComputeShader.

See Also: SetFloat, SetFloats, SetInt, SetInts, SetBuffer, SetMatrix, SetMatrixArray, SetTexture, SetVector, SetVectorArray.
