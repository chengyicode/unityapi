Projector.orthographicSize

Switch to Manual
public float orthographicSize;

Description 描述

Projection's half-size when in orthographic mode.

This is half of the vertical size of the projection volume. Horizontal projection size varies depending on aspectRatio. Orthographic size is ignored when projection is not orthographic (see orthographic). See Also: projector component.

using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        Projector proj = GetComponent<Projector>();
        proj.orthographic = true;
        proj.orthographicSize = 2.0f;
    }
}
