AndroidJavaObject.Set

public void Set(string fieldName, FieldType val);

Parameters

fieldNameThe name of the field (e.g. int counter; would have fieldName = "counter").
valThe value to assign to the field. It has to match the field type.

Description 描述

Set the value of a field in an object (non-static).

The generic parameter determines the field type.

using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        AndroidJavaObject javaObject = new AndroidJavaObject("android.text.format.Time");
        javaObject.Set<bool>("allDay", true);
    }
}
