MatchInfo.domain

Obsolete public int domain;

Description 描述

The numeric domain for the match.

This is a number, chosen by the developer that can be used to silo use of the MatchMaker within a single game. For instance if app domain 1 creates a match, that match will only appear to NetworkMatch.ListMatches requests with domain 1. This can be useful to handle different versions of a single game that may not be able to play with each other.
