EditorBuildSettings.RemoveConfigObject

public static bool RemoveConfigObject(string name);

Parameters

nameThe name in string format of the config object reference to be removed. This is the name given to the object when the reference is first created. Note: This may be different than the object name as an object can be added multiple times with different names.

Returns

bool Returns true if the reference was found and removed, otherwise false.

Description 描述

Remove a config object reference by name.
