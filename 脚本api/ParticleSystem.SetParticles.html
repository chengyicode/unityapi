ParticleSystem.SetParticles

Switch to Manual
public void SetParticles(out Particle[] particles);
public void SetParticles(out Particle[] particles, int size);
public void SetParticles(out Particle[] particles, int size, int offset);

Parameters

particlesInput particle buffer, containing the desired particle state.
sizeThe number of elements in the particles array that is written to the Particle System.
offsetThe offset into the destination particle list, to assign these particles.

Description 描述

Sets the particles of this Particle System.

Setting the lifetime of a particle to a negative value will result in that particle being removed from the Particle System. See Also: GetParticles.
