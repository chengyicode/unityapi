SupportedHeadTracking

enumeration

Description 描述

Resolution dialog setting.

Use this enum to set head tracking support in VRDaydream. See Also: PlayerSettings.VRDaydream.minimumSupportedHeadTracking, PlayerSettings.VRDaydream.maximumSupportedHeadTracking, PlayerSettings.

Properties 属性

ThreeDoFDevices that support 3 Degrees of Freedom head tracking.
SixDoFDevices that support 6 Degrees of Freedom head tracking.
