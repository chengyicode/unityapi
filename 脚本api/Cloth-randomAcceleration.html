Cloth.randomAcceleration

Switch to Manual
public Vector3 randomAcceleration;

Description 描述

A random, external acceleration applied to the cloth.

Use this to simulate randomly changing forces on the cloth, such as wind turbulences waving a flag. See Also: Cloth.externalAcceleration.

using UnityEngine;

public class Example : MonoBehaviour
{
    // Simulate wind going trough the X axis.
    void Start()
    {
        GetComponent<Cloth>().randomAcceleration = new Vector3(10, 0, 0);
    }
}
