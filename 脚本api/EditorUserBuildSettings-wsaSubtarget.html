EditorUserBuildSettings.wsaSubtarget

public static WSASubtarget wsaSubtarget;

Description 描述

Sets and gets target device type for the application to run on when building to Windows Store platform.

Default is to target "any device", which results in poorer settings choices for any particular device type but works on all of them. Choosing any particular device type will optimize settings for that device type, but may cause application to not function properly when run on another device type.
