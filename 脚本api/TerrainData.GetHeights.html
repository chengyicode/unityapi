TerrainData.GetHeights

public float[,] GetHeights(int xBase, int yBase, int width, int height);

Parameters

xBaseFirst x index of heightmap samples to retrieve.
yBaseFirst y index of heightmap samples to retrieve.
widthNumber of samples to retrieve along the heightmap's x axis.
heightNumber of samples to retrieve along the heightmap's y axis.

Description 描述

Get an array of heightmap samples.

Returns a two dimensional array of heightmap samples. The samples are represented as float values ranging from 0 to 1. The array has the dimensions [height,width] and is indexed as [y,x].
