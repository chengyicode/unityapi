GlobalObjectId.GetGlobalObjectIdSlow

public static GlobalObjectId GetGlobalObjectIdSlow(Object targetObject);

Parameters

targetObjectThe Object to be converted.

Returns

GlobalObjectId The converted GlobalObjectId. If the conversion is unsuccessful, the GlobalObjectId is set to the default null ID: "GlobalObjectId_V1-0-00000000000000000000000000000000-0-0".

Description 描述

Converts an Object reference to a GlobalObjectId.

This call is slow and should be avoided or used sparingly.
