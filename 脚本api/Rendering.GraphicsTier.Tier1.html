GraphicsTier.Tier1

Description 描述

The first graphics tier (Low) - corresponds to shader define UNITY_HARDWARE_TIER1.

This tier is selected for: - Android devices that don't have OpenGL ES 3 support - iPhone 5, 5C and earlier - iPod Touch 5th generation and earlier - iPad 4th generation and earlier - iPad Mini 1st generation - DirectX 9 class hardware on desktops - HoloLens
