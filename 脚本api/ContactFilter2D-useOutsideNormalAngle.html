ContactFilter2D.useOutsideNormalAngle

public bool useOutsideNormalAngle;

Description 描述

Sets the contact filter to filter within the minNormalAngle and maxNormalAngle range, or outside that range.

useOutsideNormalAngle uses the range between minNormalAngle and maxNormalAngle to filter the normal angle. Set useOutsideNormalAngle to true to only return normal angle outside that range. Set useOutsideNormalAngle to false to only return normal angle within that range. It is false by default.

See Also: minNormalAngle, maxNormalAngle & useNormalAngle.
