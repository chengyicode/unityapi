Texture2DArray.SetPixels

Switch to Manual
public void SetPixels(Color[] colors, int arrayElement, int miplevel);

Parameters

colorsAn array of pixel colors.
arrayElementThe texture array element index.
miplevelThe mip level.

Description 描述

Set pixel colors for the whole mip level.

This function takes a color array and changes the pixel colors of the whole mip level of the texture. Call Apply to actually upload the changed pixels to the graphics card.

See Also: SetPixels32, GetPixels, Texture2D.SetPixels, Graphics.CopyTexture.
