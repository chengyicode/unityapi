Matrix4x4.Ortho

public static Matrix4x4 Ortho(float left, float right, float bottom, float top, float zNear, float zFar);

Description 描述

Creates an orthogonal projection matrix.

The returned matrix, when used as a Camera's projection matrix, creates a view showing the area between left, right, top and bottom, with zNear and zFar as the near and far depth clipping planes.

See Also: GL.LoadPixelMatrix, GL.LoadProjectionMatrix, GUI.matrix.
