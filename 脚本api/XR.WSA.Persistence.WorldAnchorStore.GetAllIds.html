WorldAnchorStore.GetAllIds

public string[] GetAllIds();

Returns

string[] An array of string identifiers.

Description 描述

Gets all of the identifiers of the currently persisted WorldAnchors.

See Also: WorldAnchorStore.Save.

public int GetAllIds(string[] ids);

Parameters

idsA target array to receive the identifiers of the currently persisted world anchors.

Returns

int The number of identifiers stored in the target array.

Description 描述

Gets all of the identifiers of the currently persisted WorldAnchors.

If the target array is not large enough to contain all the identifiers, then only those identifiers that fit within the array will be stored and the return value will equal the size of the array. You can detect this condition by checking for a return value less than WorldAnchorStore.anchorCount. See Also: WorldAnchorStore.Save.
