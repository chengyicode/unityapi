PlayerSettings.iOS.SetiPadLaunchScreenType

Switch to Manual
public static void SetiPadLaunchScreenType(iOSLaunchScreenType type);

Description 描述

Sets the mode which will be used to generate the app's launch screen Interface Builder (.xib) file for iPad.

The launch screen image defaults to an default colored image or it can be specified using the PlayerSettings.iOS.SetLaunchScreenImage function.
