InputTracking.GetTrackedNodeStates(List<XR.XRNodeState>)

Parameters

nodeStatesA list that is populated with XRNodeState objects.

Description 描述

Describes all currently connected XRNodes and provides available tracking states for each.

Not all XR platforms provide complete tracking data. Use the properties XR.XRNodeState.positionAvailable, XR.XRNodeState.rotationAvailable, etc. to determine whether specific data is available for a tracked node. Also, this method will clear any members of the List passed before populating it.
