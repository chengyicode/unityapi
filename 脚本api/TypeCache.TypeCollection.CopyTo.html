TypeCache.TypeCollection.CopyTo

public void CopyTo(Type[] array, int arrayIndex);
public void CopyTo(Array array, int arrayIndex);

Parameters

arrayThe one-dimensional destination Array to which elements are copied from TypeCollection.
arrayIndexThe zero-based index where elements are copied to in the destination Array.

Description 描述

Copies the collection to an array.

The destination array should contain enough space to fit the collection. The destination array must be of type Type[].
