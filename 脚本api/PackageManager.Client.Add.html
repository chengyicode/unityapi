Client.Add

public static PackageManager.Requests.AddRequest Add(string identifier);

Parameters

identifierA string representing the package to be added. To install a specific version of a package, use a package identifier (name@version). To install the latest compatible version of a package, use the package name only. To install a git package, specify a git url.

Returns

AddRequest An AddRequest instance, which you can use to monitor the asynchronous operation, and when complete, get the result.

Description 描述

Adds a package dependency to the project.
