Description 描述

Unity Package Manager client class.

Use the Client class to manage the packages used in a project.

Static Methods 静态方法

AddAdds a package dependency to the project.
EmbedEmbeds a package inside the project.
ListLists the packages the project depends on.
RemoveRemoves a previously added package from the project.
ResetToEditorDefaultsResets the list of packages installed for this project to the editor's default configuration. This operation will clear all packages added to the project and keep only the packages set for the current editor default configuration.
SearchSearches the Unity package registry for the given package.
SearchAllSearches the Unity package registry for all packages compatible with the current Unity version.
