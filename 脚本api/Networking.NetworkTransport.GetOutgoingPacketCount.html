NetworkTransport.GetOutgoingPacketCount

Obsolete public static int GetOutgoingPacketCount();

Returns

int Packets count sent from networking library start (from call Networking.NetworkTransport.Init) for all hosts.

Description 描述

Returns how many packets have been sent from start (from call Networking.NetworkTransport.Init) for all hosts.

To determine the packet rate, sample this value periodically and divide the difference by the intervening time period.
