AnimatorStateMachine.AddAnyStateTransition

public Animations.AnimatorStateTransition AddAnyStateTransition(Animations.AnimatorState destinationState);
public Animations.AnimatorStateTransition AddAnyStateTransition(Animations.AnimatorStateMachine destinationStateMachine);

Parameters

destinationStateThe destination state.
destinationStateMachineThe destination statemachine.

Description 描述

Utility function to add an AnyState transition to the specified state or statemachine.

The transition asset that is created is added as a sub asset of the state machine. Its important that AnyStateTransitions are added to the root state machine. AnyStateTranistions added to a sub state machine will be discarded at runtime. This function pushes an Undo operation.
