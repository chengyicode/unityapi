LineRenderer.GetPositions

Switch to Manual
public int GetPositions(out Vector3[] positions);

Parameters

positionsThe array of positions to retrieve. The array passed should be of at least positionCount in size.

Returns

int How many positions were actually stored in the output array.

Description 描述

Get the positions of all vertices in the line.

This method is preferred to GetPosition when retrieving all positions, as it is more efficient to get all positions using a single command than to get each position individually.

See Also: positionCount property, GetPosition function.
