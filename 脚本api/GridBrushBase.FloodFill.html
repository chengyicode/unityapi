GridBrushBase.FloodFill

public void FloodFill(GridLayout gridLayout, GameObject brushTarget, Vector3Int position);

Parameters

gridLayout Grid used for layout.
brushTargetTargets of flood fill operation. By default the currently selected GameObject.
positionStarting position of the flood fill.

Description 描述

Flood fills data onto a grid given the starting coordinates of the cell.

Flood fill all the cells that are logically connected with the starting position.
