RemoteSettings.GetFloat

public static float GetFloat(string key, float defaultValue = 0.0F);

Parameters

keyThe key identifying the setting.
defaultValueThe default value to use if the setting identified by the key parameter cannot be found or is unavailable.

Returns

float The current value of the setting identified by key, or the default value.

Description 描述

Gets the value corresponding to remote setting identified by key, if it exists.

If it doesn't exist, it will return defaultValue. If you don't provide a default value, it will return 0.0.

using UnityEngine;

public class HandleRemoteSettings : MonoBehaviour
{
    private void Start()
    {
        Debug.Log(RemoteSettings.GetFloat("gameBaseAcceleration"));
    }
}
