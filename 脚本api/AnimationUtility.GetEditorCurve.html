AnimationUtility.GetEditorCurve

Obsolete public static AnimationCurve GetEditorCurve(AnimationClip clip, string relativePath, Type type, string propertyName);
public static AnimationCurve GetEditorCurve(AnimationClip clip, EditorCurveBinding binding);

Description 描述

Return the float curve that the binding is pointing to.

Unity automatic combines position curves, scale curves, rotation curves internally.

Thus transform curves always get combined and their keyframes is always the union of all keyframe points. In the Editor AnimationClip's let you specify special editor curves which do not get combined, thus it lets the user edit curves in a more intuitive way.

See also: AnimationUtility.GetObjectReferenceCurve.
