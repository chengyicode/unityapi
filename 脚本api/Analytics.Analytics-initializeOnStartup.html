Analytics.initializeOnStartup

public static bool initializeOnStartup;

Description 描述

Reports whether Unity is set to initialize Analytics on startup.

If false, Unity does not initialize Analytics until you call Analytics.ResumeInitialization. Setting this property to false at runtime (rather than in an Editor script) pauses Analytics initialization. Use Analytics.ResumeInitialization to resume initialization. Setting InitializeOnStartup back to true does not resume initialization.

By default InitializeOnStartup is true.

See Also: Analytics.AnalyticsSettings.InitializeOnStartup, Analytics.ResumeInitialization RuntimeInitializeOnLoadMethodAttribute
