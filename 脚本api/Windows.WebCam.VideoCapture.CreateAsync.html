VideoCapture.CreateAsync

public static void CreateAsync(Windows.WebCam.VideoCapture.OnVideoCaptureResourceCreatedCallback onCreatedCallback);
public static void CreateAsync(bool showHolograms, Windows.WebCam.VideoCapture.OnVideoCaptureResourceCreatedCallback onCreatedCallback);

Parameters

showHologramsWill allow you to capture holograms in your video.
onCreatedCallbackThis callback will be invoked when the VideoCapture instance is created and ready to be used.

Description 描述

Asynchronously creates an instance of a VideoCapture object that can be used to record videos from the web camera to disk.

If the instance failed to be created, the instance returned will be null. Capturing holograms is currently disabled therefore the showHolograms parameter will be ignored.
