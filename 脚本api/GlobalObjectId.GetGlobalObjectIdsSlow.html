GlobalObjectId.GetGlobalObjectIdsSlow

public static void GetGlobalObjectIdsSlow(Object[] objects, out GlobalObjectId[] outputIdentifiers);

Parameters

objectsArray of Objects to convert.
outputIdentifiersResulting array of GlobalObjectIds.

Description 描述

Creates an array of GlobalObjectIds based on an array of Objects.

This call is slow and should be avoided or used sparingly.
