iOSLaunchScreenType

enumeration

Description 描述

iOS launch screen settings.

Properties 属性

DefaultUse the default launch screen (dark blue background).
ImageAndBackgroundRelativeUse a custom launch screen image specified in the iOS Player Settings or with PlayerSettings.iOS.SetLaunchScreenImage which will be scaled across the entire screen.
CustomXibUse a specified custom Interface Builder (.xib) file in Player Settings.
NoneGenerate the Xcode project without any custom launch screens.
ImageAndBackgroundConstantUse a custom launch screen image specified in the iOS Player Settings or with PlayerSettings.iOS.SetLaunchScreenImage and use its original dimensions.
