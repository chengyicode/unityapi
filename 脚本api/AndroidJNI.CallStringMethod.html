AndroidJNI.CallStringMethod

public static string CallStringMethod(IntPtr obj, IntPtr methodID, jvalue[] args);

Description 描述

Calls a Java instance method defined by methodID, optionally passing an array of arguments (args) to the method.

This is a convenience function that calls CallObjectMethod() with the same parameters, but creates a managed string from the result.

See Also: AndroidJNI.CallObjectMethod.
