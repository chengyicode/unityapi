AlbedoSwatchInfo

struct in UnityEditor.Rendering

/

Implemented in:UnityEditor

Description 描述

Contains the custom albedo swatch data.

The custom AlbedoSwatchInfo struct contains custom data that can be used to specify an albedo value to validate the albedo values in the Scene. For details on how to use the data, see EditorGraphicsSettings.albedoSwatches.

Properties 属性

colorColor of the albedo swatch that is shown in the physically based rendering validator user interface.
maxLuminanceThe maximum luminance value used to validate the albedo for the physically based rendering albedo validator.
minLuminanceThe minimum luminance value used to validate the albedo for the physically based rendering albedo validator.
nameName of the albedo swatch to show in the physically based renderer validator user interface.
