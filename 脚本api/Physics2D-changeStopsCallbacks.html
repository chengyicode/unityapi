Physics2D.changeStopsCallbacks

Obsolete public static bool changeStopsCallbacks;

Description 描述

Whether or not to stop reporting collision callbacks immediately if any of the objects involved in the collision are deleted/moved.

During a collision callback, if either a Collider2D or Rigidbody2D that is taking part in that particular callback is deleted or moved then this property controls whether callbacks will continue for this contact or not.

See Also: Collider2D, Rigidbody2D.
