ILocalUser

interface in UnityEngine.SocialPlatforms


Implements interfaces:IUserProfile

Description 描述

Represents the local or currently logged in user.

Properties 属性

authenticatedChecks if the current user has been authenticated.
friendsThe users friends list.
underageIs the user underage?

Public Methods 公共方法

AuthenticateAuthenticate the local user to the current active Social API implementation and fetch his profile data.
LoadFriendsFetches the friends list of the logged in user. The friends list on the Social.localUser instance is populated if this call succeeds.
