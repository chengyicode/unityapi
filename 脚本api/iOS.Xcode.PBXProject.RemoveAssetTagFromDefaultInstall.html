PBXProject.RemoveAssetTagFromDefaultInstall

public void RemoveAssetTagFromDefaultInstall(string targetGuid, string tag);

Parameters

targetGuidThe GUID of the target as returned by [[TargetGuidByName()]].
tagThe name of the asset tag.

Description 描述

Removes the asset tag from the list of tags to download during initial installation.

The function does nothing if the tag is not already configured for downloading during initial installation.
