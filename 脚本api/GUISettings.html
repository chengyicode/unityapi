Description 描述

General settings for how the GUI behaves.

These are shared by all elements in a GUISkin.

Properties 属性

cursorColorThe color of the cursor in text fields.
cursorFlashSpeedThe speed of text field cursor flashes.
doubleClickSelectsWordShould double-clicking select words in text fields.
selectionColorThe color of the selection rect in text fields.
tripleClickSelectsLineShould triple-clicking select whole text in text fields.
