AndroidJNI.GetByteArrayElement

Obsolete public static byte GetByteArrayElement(IntPtr array, int index);

Description 描述

Returns the value of one element of a primitive array.

This function is a special case of GetByteArrayRegion(), called with region size set to 1.

AndroidJNI.GetByteArrayElement is obsolete. Use AndroidJNI.GetSByteArrayElement instead.
