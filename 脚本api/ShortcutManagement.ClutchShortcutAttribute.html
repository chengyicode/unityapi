Description 描述

Attribute used to register a static method as the action for a clutch shortcut.

A clutch shortcut is triggered when the binding for the shortcut is pressed down and also when it is released. The method on which this attribute is placed must take a single argument of type ShortcutArguments.

Constructors

ClutchShortcutAttributeCreates an attribute for a clutch shortcut with an ID, optional context, and optional default binding.
