ShortcutArguments.stage

public ShortcutManagement.ShortcutStage stage;

Description 描述

The stage at which a shortcut action was invoked.

This is used to determine if the action of a clutch shortcut was invoked due to pressing the binding (ShortcutManagement.ShortcutState.Begin) or releasing the binding (ShortcutManagement.ShortcutState.End).
