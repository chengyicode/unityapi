GestureRecognizer.NavigationUpdatedEventDelegate

public delegate void NavigationUpdatedEventDelegate(XR.WSA.Input.InteractionSourceKind source, Vector3 normalizedOffset, Ray headRay);

Parameters

sourceIndicates which input medium triggered this event.
normalizedOffsetThe last known normalized offset, since the navigation gesture began, of the input within the unit cube for the navigation gesture.
headRayRay (with normalized direction) from user at the time this gesture began.

Description 描述

Callback indicating a update event.

Users should register with NavigationUpdatedEvent.
