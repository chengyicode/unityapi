GeometryChangedEvent.GetPooled

public static UIElements.GeometryChangedEvent GetPooled(Rect oldRect, Rect newRect);

Parameters

oldRectThe old dimensions of the element.
newRectThe new dimensions of the element.

Returns

GeometryChangedEvent Returns an initialized event.

Description 描述

Gets an event from the event pool and initializes the event with the specified values. Use this method instead of instancing new events. Use Dispose() to release events back to the event pool.
