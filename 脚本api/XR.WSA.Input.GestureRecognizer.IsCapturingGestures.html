GestureRecognizer.IsCapturingGestures

public bool IsCapturingGestures();

Returns

bool True if the GestureRecognizer is receiving events or false otherwise.

Description 描述

Used to query if the GestureRecognizer is currently receiving Gesture events.

The GestureRecognizer will receive events after StartCapuringGestures is called and stop when StopCapturingGestures or CancelGestures is called.
