PaintContext Constructor

public PaintContext(Terrain terrain, RectInt pixelRect, int targetTextureWidth, int targetTextureHeight);

Parameters

terrainTerrain that defines terrain space for this PaintContext.
pixelRectPixel rectangle to edit in the target terrain texture.
targetTextureWidthWidth of the target terrain texture (per Terrain).
targetTextureHeightHeight of the target terrain texture (per Terrain).

Description 描述

Creates a new PaintContext, to edit a target texture on a Terrain, in a region defined by pixelRect.

This constructor finds all Terrains that touch the pixelRect, searching across adjacent connected Terrains. It also calculates the relevant regions on each Terrain, as well as the transforms between them.
