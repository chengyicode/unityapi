AudioRenderer

class in UnityEngine

/

Implemented in:UnityEngine.AudioModule

Description 描述

Allow recording the main output of the game or specific groups in the AudioMixer.

Static Methods 静态方法

GetSampleCountForCaptureFrameReturns the number of samples available since the last time AudioRenderer.Render was called. This is dependent on the frame capture rate.
RenderPerforms the recording of the main output as well as any optional mixer groups that have been registered via AudioRenderer.AddMixerGroupSink.
StartEnters audio recording mode. After this Unity will output silence until AudioRenderer.Stop is called.
StopExits audio recording mode. After this audio output will be audible again.
