Class used to access properties of the Oculus VR device.

Properties of this class are shared across platforms.

Static Properties

dashSupportEnable Oculus Dash support for the application.
lowOverheadModeEnable Low Overhead driver optimizations for Oculus.
protectedContextEnable protected graphics context for Oculus.
sharedDepthBufferEnable Shared Depth Buffer support with Oculus.
v2SigningEnable application signing with the Android Package (APK) Signature Scheme v2.
