RenderSettings.useRadianceAmbientProbe

public static bool useRadianceAmbientProbe;

Description 描述

If enabled, ambient trilight will be sampled using the old radiance sampling method.

This feature is here to maintain backwards compatibility for realtime ambient light when using the trilight mode. The new default behavior now uses the correct sampling behavior. Please note that the new sampling method is slower.
