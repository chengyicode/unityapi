ImportAssetOptions

enumeration

Description 描述

Asset importing options.

See Also: AssetDatabase.ImportAsset, AssetDatabase.Refresh.

Properties 属性

DefaultDefault import options.
ForceUpdateUser initiated asset import.
ForceSynchronousImportImport all assets synchronously.
ImportRecursiveWhen a folder is imported, import all its contents as well.
DontDownloadFromCacheServerForce a full reimport but don't download the assets from the cache server.
ForceUncompressedImportForces asset import as uncompressed for edition facilities.
