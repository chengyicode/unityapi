PlayableExtensions.SetPlayState

Obsolete public static void SetPlayState(U playable, Playables.PlayState value);

Parameters

playableThe Playable used by this operation.

Description 描述

Changes the current PlayState of the Playable.

NOTE: This is a templated extension method that can be used on any type that inherits from IPlayable

PlayState.Playing: The local time of this Playable will be updated during the evaluation of the PlayableGraph.

PlayState.Paused: The local time of this Playable will not be updated during the evaluation of the PlayableGraph.
