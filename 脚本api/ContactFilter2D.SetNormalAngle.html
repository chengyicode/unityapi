ContactFilter2D.SetNormalAngle

public void SetNormalAngle(float minNormalAngle, float maxNormalAngle);

Parameters

minNormalAngleThe value used to set the minNormalAngle.
maxNormalAngleThe value used to set the maxNormalAngle.

Description 描述

Sets the minNormalAngle and maxNormalAngle filter properties and turns on normal angle filtering by setting useNormalAngle to true.
