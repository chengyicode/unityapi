TreeView.AddExpandedRows

protected void AddExpandedRows(IMGUI.Controls.TreeViewItem root, IList<TreeViewItem> rows);

Parameters

rootRoot of the TreeView.
rowsRows that will be refilled using the expanded state of TreeView.

Description 描述

Adds the expanded rows of the full tree to the input list. Only use this method if a full tree was built in BuildRoot.

Note that the rows list will not be cleared before adding items.

See Also: BuildRoot, BuildRows.
