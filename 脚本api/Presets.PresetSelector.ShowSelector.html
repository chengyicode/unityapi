PresetSelector.ShowSelector

public static void ShowSelector(Object[] targets, Presets.Preset currentSelection, bool createNewAllowed);
public static void ShowSelector(Object target, Presets.Preset currentSelection, bool createNewAllowed, Presets.PresetSelectorReceiver eventReceiver);

Parameters

targetsList of objects to which the selected Preset is applied. DefaultPresetSelectorReceiver applies the Preset selection.
currentSelectionThe selected Preset when the window is opened. Set to 'null' for no selection.
createNewAllowedSet to true to show the 'Save current to...' button. Set to false to hide this button.
targetObject that identifies the type of Preset asset being selected. The modal window filters the selector view based on this object.
eventReceiverThe PresetSelectorReceiver instance that the PresetSelector uses to send events.

Description 描述

Opens a modal window for selecting a Preset.
