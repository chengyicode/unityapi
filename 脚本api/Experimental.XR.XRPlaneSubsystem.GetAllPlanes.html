XRPlaneSubsystem.GetAllPlanes

public void GetAllPlanes(List<BoundedPlane> planesOut);

Parameters

planesOutA list of BoundedPlanes containing all planes currently tracked by the system.

Description 描述

Get all the BoundedPlanes currently tracked by the system.

Existing data is replaced with a list of the currently tracked planes. To avoid excessive heap allocations, developers may reuse the same list each call. A reallocation only happens when the list's capacity is insufficient.

If planesOut is null, this method throws ArgumentNullException.
