TreeView.GetFoldoutIndent

protected float GetFoldoutIndent(IMGUI.Controls.TreeViewItem item);

Parameters

itemItem used to determine the indent.

Returns

float Indent for the foldout arrow.

Description 描述

Returns the horizontal foldout offset for an item. This is where the foldout arrow is rendered.

Use this when overriding TreeView.RowGUI to make custom GUI content for the item.

See Also: GetContentIndent.
