GlobalObjectId.TryParse

public static bool TryParse(string stringValue, out GlobalObjectId id);

Parameters

stringValueThe string representation of a GlobalObjectId.
Example: "GlobalObjectId_V1-2-74c253e3f16be4776bb2d88e01f77c8a-902906726-0".
idThe GlobalObjectId struct for the parsed values.

Returns

bool Returns true if the string representation is successfully parsed. Otherwise, returns false.

Description 描述

Parses the string representation of a GlobalObjectId into a GlobalObjectId struct.
