PBXProject.TargetGuidByName

public string TargetGuidByName(string name);

Parameters

nameThe name of the native target.

Returns

string The GUID identifying the native target.

Description 描述

Returns the GUID of the native target with the given name.

In projects produced by Unity the main target can be retrieved via GetUnityTargetName, whereas the test target name can be retrieved by GetUnityTestTargetName.
