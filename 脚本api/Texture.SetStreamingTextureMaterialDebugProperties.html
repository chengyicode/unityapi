Texture.SetStreamingTextureMaterialDebugProperties

public static void SetStreamingTextureMaterialDebugProperties();

Description 描述

Uploads additional debug information to materials using textures set to stream mip maps.

This sets up debug shader constants for all materials which use streaming textures. This sets up a constant called MainTex_mipInfo which contains the following 4 values : Quality Settings Min Streaming Mip Level, Texture Original Mip map Count, Desired Streaming Mip Level, Loaded Mip Level.

Camera.SetReplacementShader can be used to set a custom shader for a camera to render debugging information.
