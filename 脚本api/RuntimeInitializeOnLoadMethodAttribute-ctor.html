RuntimeInitializeOnLoadMethodAttribute Constructor

public RuntimeInitializeOnLoadMethodAttribute();
public RuntimeInitializeOnLoadMethodAttribute(RuntimeInitializeLoadType loadType);

Parameters

loadTypeDetermine whether methods are called before or after the Scene is loaded.

Description 描述

Creation of the runtime class used when Scenes are loaded.

See Also: RuntimeInitializeOnLoadMethodAttribute, RuntimeInitializeLoadType.

// Demonstration of RuntimeInitializeOnLoadMethod and the argument it can take.
using UnityEngine;

class MyClass
{
    [RuntimeInitializeOnLoadMethod(RuntimeInitializeLoadType.BeforeSceneLoad)]
    static void OnBeforeSceneLoadRuntimeMethod()
    {
        Debug.Log("Before first Scene loaded");
    }

    [RuntimeInitializeOnLoadMethod(RuntimeInitializeLoadType.AfterSceneLoad)]
    static void OnAfterSceneLoadRuntimeMethod()
    {
        Debug.Log("After first Scene loaded");
    }

    [RuntimeInitializeOnLoadMethod]
    static void OnRuntimeMethodLoad()
    {
        Debug.Log("RuntimeMethodLoad: After first Scene loaded");
    }
}
