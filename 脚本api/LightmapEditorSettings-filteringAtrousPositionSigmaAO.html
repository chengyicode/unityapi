LightmapEditorSettings.filteringAtrousPositionSigmaAO

public static float filteringAtrousPositionSigmaAO;

Description 描述

Specifies the threshold the Progressive lightmapper uses to filter ambient occlusion stored in the lightmap when using A-Trous filter. The default value is 1. The value range is 0 to 2.

A higher value will increase the threshold, reducing noise in the direct layer of the lightmap. Too high of a value can cause a loss of detail in the lightmap.
