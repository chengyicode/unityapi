Description 描述

Describes supported file type for File Type Association declaration.

See Also: WSAFileTypeAssociations.

Properties 属性

contentTypeThe 'Content Type' value for the file type's MIME content type. For example: 'image/jpeg'. Can also be left blank.
fileTypeFile type extension. For ex., .myUnityGame
