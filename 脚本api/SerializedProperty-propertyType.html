SerializedProperty.propertyType

public SerializedPropertyType propertyType;

Description 描述

Type of this property (Read Only).

Property type determines which of the "value" variable accessors are valid. For example, only boolValue is valid for SerializedPropertyType.Boolean

Both intValue and longValue are valid for SerializedPropertyType.Integer. 
Both floatValue and doubleValue are valid for SerializedPropertyType.Float.

See Also: SerializedPropertyType.
