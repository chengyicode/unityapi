LightmapEditorSettings.denoiserTypeAO

public static LightmapEditorSettings.DenoiserType denoiserTypeAO;

Description 描述

The type of denoising applied to the ambient occlusion for the lightmap.

Use denoising to remove visible noise from your baked ambient occlusion for the lightmap. This can reduce the number of samples needed, light leaking, and loss of detail.
