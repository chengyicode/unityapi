Provider.Lock

public static VersionControl.Task Lock(VersionControl.AssetList assets, bool locked);
public static VersionControl.Task Lock(VersionControl.Asset asset, bool locked);

Parameters

assetsList of assets to lock/unlock.
lockedTrue to lock assets, false to unlock assets.
assetAsset to lock/unlock.

Description 描述

Attempt to lock an asset for exclusive editing.

Depening on version control system and server setup this might be an exclusive lock preventing other people from even modifing the asset or or it might simple prevent other people from submitting the asset to the version control server.
