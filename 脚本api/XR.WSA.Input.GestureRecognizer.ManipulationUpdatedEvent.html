GestureRecognizer.ManipulationUpdatedEvent

Parameters

valueDelegate to be triggered when a updated event is triggered.

Description 描述

Fires when a Manipulation gesture is updated due to hand movement.

This event fires when the relevant hand that had a ManipulationStarted updates its position. This event does not fire for voice interactions and controllers.

ManipulationUpdatedEvent has been deprecated. Use ManipulationUpdated instead.
