XRReferencePointSubsystem.GetAllReferencePoints

public void GetAllReferencePoints(List<ReferencePoint> referencePointsOut);

Parameters

referencePointsOutA list of ReferencePoints containing all reference points.

Description 描述

Retrieves all ReferencePoints added by calls to XRReferencePointSubsystem.TryAddReferencePoint.

Existing data is replaced with a list of the current reference points. To avoid excessive heap allocations, developers may reuse the same list each call. A reallocation only happens when the list's capacity is insufficient.

If referencePointsOut is null, this method throws ArgumentNullException.
