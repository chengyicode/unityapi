CustomGridBrushAttribute

class in UnityEngine

/

Implemented in:UnityEngine.TilemapModule

Description 描述

Attribute to define the class as a grid brush and to make it available in the palette window.

Properties 属性

defaultBrushIf set to true, brush will replace Unity built-in brush as the default brush in palette window.Only one class at any one time should set defaultBrush to true.
defaultNameName of the default instance of this brush.
hideAssetInstancesHide all asset instances of this brush in the tile palette window.
hideDefaultInstanceHide the default instance of brush in the tile palette window.

Constructors
