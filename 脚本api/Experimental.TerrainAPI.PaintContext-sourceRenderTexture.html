PaintContext.sourceRenderTexture

public RenderTexture sourceRenderTexture;

Description 描述

(Read Only) Render target that stores the original data from the Terrains.

This RenderTexture contains all of the data collected from all Terrains that intersect the PaintContext. The RenderTexture is created by PaintContext.CreateRenderTargets, and populated by one of the Gather functions (PaintContext.GatherHeightmap, PaintContext.GatherAlphamap or PaintContext.GatherNormals).

See Also: PaintContext, PaintContext.destinationRenderTexture.
