NavMesh.CalculateTriangulation

public static AI.NavMeshTriangulation CalculateTriangulation();

Description 描述

Calculates triangulation of the current navmesh.

Calculates and returns a simple triangulation of the current navmesh - containing the vertices, triangle indices and navmesh layers.

The returned mesh contains only the triangles used for pathfinding. It does not contain the detail that is used to place the agents on the walkable surface. This can be noticeable on locations with curved surfaces.
