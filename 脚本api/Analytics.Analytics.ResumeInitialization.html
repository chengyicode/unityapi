Analytics.ResumeInitialization

public static Analytics.AnalyticsResult ResumeInitialization();

Description 描述

Resume Analytics initialization.

By default, Unity initializes Analytics automatically at app startup, which involves sending data to the Unity Analytics service. To prevent or delay initialization, set Analytics.AnalyticsSettings.InitializeOnStartup to false in an Editor script. You can also set Analytics.Analytics.InitializeOnStartup false at runtime, in which case Unity halts initialization in its current state until you call ResumeInitialization.

See Also: Analytics.AnalyticsSettings.InitializeOnStartup, Analytics.Analytics.InitializeOnStartup
