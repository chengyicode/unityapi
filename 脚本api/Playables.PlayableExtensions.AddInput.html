PlayableExtensions.AddInput

public static int AddInput(U playable, V sourcePlayable, int sourceOutputIndex, float weight);

Parameters

playableThe Playable used by this operation.
sourcePlayableThe Playable to connect to.
sourceOutputIndexThe output port of the Playable.
weightThe weight of the created input port.

Returns

int The index of the newly created input port.

Description 描述

Create a new input port and connect it to the output port of the given Playable.

If several inputs are needed, it is better to use SetInputCount and ConnectInput.
This method is a mere helper on top of PlayableGraph.Connect.

NOTE: This is a templated extension method that can be used on any type that inherits from IPlayable.
