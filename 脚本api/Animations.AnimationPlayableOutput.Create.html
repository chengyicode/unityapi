AnimationPlayableOutput.Create

public static Animations.AnimationPlayableOutput Create(Playables.PlayableGraph graph, string name, Animator target);

Parameters

graphThe PlayableGraph that will contain the AnimationPlayableOutput.
nameThe name of the output.
targetThe Animator that will process the PlayableGraph.

Returns

AnimationPlayableOutput A new AnimationPlayableOutput attached to the PlayableGraph.

Description 描述

Creates an AnimationPlayableOutput in the PlayableGraph.

The Animator plays the source Playable of the AnimationPlayableOutput. This source Playable can be set with SetSourcePlayable.
