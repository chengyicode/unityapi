Lightmapping.probesIgnoreDirectEnvironment

public static bool probesIgnoreDirectEnvironment;

Description 描述

If enabled ignores the direct contribution from the environment lighting in baked probes.

This feaure is useful in combination with custom shaders where the sky contribution is added directly in the shader. This way various per pixel sky occlusion techniques can be integrated. NOTE: Only exposed for the Progressive Lightmapper baking backend.
