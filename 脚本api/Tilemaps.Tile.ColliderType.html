Description 描述

Enum for determining what collider shape is generated for this Tile by the TilemapCollider2D.

枚举用于确定什么是这个生成碰撞器形状Tile由TilemapCollider2D。

Properties

空 None TilemapCollider2D不会为Tile生成碰撞体形状 No collider shape is generated for the Tile by the TilemapCollider2D.
精灵 Sprite Sprite轮廓被TilemapCollider2D用作Tile的碰撞器形状 The Sprite outline is used as the collider shape for the Tile by the TilemapCollider2D.
网格 Grid TilemapCollider2D将网格布局边界轮廓用作Tile的对撞机形状 The grid layout boundary outline is used as the collider shape for the Tile by the TilemapCollider2D.
