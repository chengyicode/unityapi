DragAndDropArgs

/

Implemented in:UnityEditor

Description 描述

Method arguments for the HandleDragAndDrop virtual method.

Properties 属性

dragAndDropPositionWhen dragging items the current drag can have the following 3 positions relative to the items: Upon an item, Between two items or Outside items.
insertAtIndexThis index refers to the index in the children list of the parentItem where the current drag is positioned.
parentItemThe parent item is set if the drag is either upon this item or between two of its children.
performDropThis value is false as long as the mouse button is down, when the mouse button is released it is true.
