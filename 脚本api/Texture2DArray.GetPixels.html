Texture2DArray.GetPixels

Switch to Manual
public Color[] GetPixels(int arrayElement, int miplevel);

Parameters

arrayElementArray slice to read pixels from.
miplevelMipmap level to read pixels from.

Returns

Color[] Array of pixel colors.

Description 描述

Returns pixel colors of a single array slice.

Note that using Color32 data and GetPixels32 can be faster and consume less memory.

See Also: GetPixels32, SetPixels, Graphics.CopyTexture.
