GestureRecognizer.ManipulationStartedEvent

Parameters

valueDelegate to be triggered when a started event is triggered.

Description 描述

Fires when an interaction becomes a Manipulation gesture.

This event fires when a finger is pressed and then moves outside of the small Manipulation deadzone. This event does not fire for voice interactions and controllers.

ManipulationStartedEvent has been deprecated. Use ManipulationStarted instead.
