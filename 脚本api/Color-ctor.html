Color Constructor

public Color(float r, float g, float b, float a);

Parameters

rRed component.
gGreen component.
bBlue component.
aAlpha component.

Description 描述

Constructs a new Color with given r,g,b,a components.

using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        Color newColor = new Color(0.3f, 0.4f, 0.6f, 0.3f);
    }
}
