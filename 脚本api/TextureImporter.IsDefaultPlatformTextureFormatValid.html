TextureImporter.IsDefaultPlatformTextureFormatValid

Switch to Manual
public static bool IsDefaultPlatformTextureFormatValid(TextureImporterType textureType, TextureImporterFormat currentFormat);

Parameters

currentFormatThe TextureImporterType that the importer uses.
textureTypeThe TextureImporterFormat to validate.

Returns

bool Returns true if ref::TextureImporterFormat:: is valid and can be set. Returns false otherwise.

Description 描述

Validates ref::TextureImporterFormat:: based on the type of the current format (ref::TextureImporterType::) and the default platform.

This method depends on the platforms that are available in the current Unity installation. Because some platforms do not support all formats, these formats cannot be used with the Default Settings. Use this method to check that a format is valid and can be set.
