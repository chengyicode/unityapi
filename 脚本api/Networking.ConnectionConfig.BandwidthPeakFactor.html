ConnectionConfig.BandwidthPeakFactor

Obsolete public float BandwidthPeakFactor;

Description 描述

Defines, when multiplied internally by InitialBandwidth, the maximum bandwidth that can be used under burst conditions.

If you do not send any data for some time, the bandwidth available to your app is the full bandwidth specified in InitialBandwidth. However, it is possible to occupy more than this bandwidth for a short period of time, known as a burst. This property limits the bandwidth that can be used during such a burst. This parameter can be considered to be a smoothing factor for oscillating traffic; if traffic is irregular and the peak bandwidth is high, the network may see spikes of bandwidth use leading to packets being discarded by traffic shaping or management functions elsewhere in the network. For more information, see the token bucket algorithm, in particular a discussion of burst size.
