VFXManager.maxDeltaTime

public static float maxDeltaTime;

Description 描述

The maximum allowed delta time for an update interval. This limit affects fixedDeltaTime and deltaTime. The tick rate is in seconds.

For in-game time, the fixedDeltaTime and deltaTime intervals are affected by Time.timeScale. These values are less than or equal to VFXManager.maxDeltaTime.

The fixedDeltaTime interval can be an integer multiple of VFXManager.fixedTimeStep.
