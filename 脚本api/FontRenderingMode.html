FontRenderingMode

enumeration

Description 描述

Font rendering mode constants for TrueTypeFontImporter.

Properties 属性

SmoothUse Anti-Aliased Font rendering. When using dynamic fonts, this is the mode which is fastest in rendering font textures.
HintedSmoothUse Anti-Aliased Font rendering with hinting. This forces character lines to run along pixel boundaries.
HintedRasterUse hinted font rendering without anti-aliasing. This is the crispest font rendering option, and may be most readable for small font sizes.
OSDefaultUse the OS default font rendering mode. This mode is only available for dynamic fonts, as static fonts are generated at build time.
