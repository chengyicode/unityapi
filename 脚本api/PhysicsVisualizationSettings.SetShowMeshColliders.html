PhysicsVisualizationSettings.SetShowMeshColliders

public static void SetShowMeshColliders(PhysicsVisualizationSettings.MeshColliderType colliderType, bool show);

Description 描述

Should MeshColliders be shown.

Convex and non-convex MeshColliders are controlled separately by colliderType.

See Also: MeshColliderType, MeshCollider.convex.
