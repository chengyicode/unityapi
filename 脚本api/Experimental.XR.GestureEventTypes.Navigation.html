GestureEventTypes.Navigation

Description 描述

Flag indicating a navigation gesture.

Navigation gestures operate like a virtual joystick, and you can use them to navigate UI widgets, such as radial menus. You tap and hold to start the gesture and then move your hand within a normalized 3D cube, centered around the initial press. You can move your hand along the x-axis, y-axis or z-axis from a value of -1 to 1, with 0 being the starting point.
