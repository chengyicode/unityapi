Description 描述

Specifies the type of geometry the Composite Collider generates.

See Also: CompositeCollider2D.geometryType.

Properties 属性

OutlinesSets the Composite Collider to generate closed outlines for the merged Collider geometry consisting of only edges.
PolygonsSets the Composite Collider to generate closed outlines for the merged Collider geometry consisting of convex polygon shapes.
