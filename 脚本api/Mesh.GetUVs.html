Parameters

channelThe UV channel. Indices start at 0, which corresponds to uv. Note that 1 corresponds to uv2.
uvsA list of UVs to populate.

Description 描述

Gets the UVs of the Mesh.

Gets the Mesh's UVs (texture coordinates) as a List of either Vector2, Vector3, or Vector4.
