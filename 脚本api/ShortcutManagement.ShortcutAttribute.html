Description 描述

Attribute used to register a static method as the action for an action shortcut.

An action shortcut is triggered when the binding for the shortcut is pressed down. The method on which this attribute is placed must either take no arguments or take a single argument of type ShortcutArguments.

Properties 属性

displayNameOptional override of the Shortcut ID when listing the Shortcut in the configuration interface.
