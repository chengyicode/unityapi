AnimationMode.AddPropertyModification

public static void AddPropertyModification(EditorCurveBinding binding, PropertyModification modification, bool keepPrefabOverride);

Parameters

bindingDescription of the animation clip curve being modified.
modificationObject property being modified.
keepPrefabOverrideIndicates whether to retain modifications when the targeted object is an instance of a Prefab.

Description 描述

Marks a property as currently being animated.

Used by the animation recording system to add a property to the animation mode snapshot when the user modifies it (through script, using the inspector, etc.)
