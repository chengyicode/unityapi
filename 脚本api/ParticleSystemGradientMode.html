ParticleSystemGradientMode

enumeration

Description 描述

The particle gradient mode.

This is used by MinMaxGradient to determine which mode to evaluate color gradients in.

Properties 属性

ColorUse a single color for the MinMaxGradient.
GradientUse a single color gradient for the MinMaxGradient.
TwoColorsUse a random value between 2 colors for the MinMaxGradient.
TwoGradientsUse a random value between 2 color gradients for the MinMaxGradient.
RandomColorDefine a list of colors in the MinMaxGradient, to be chosen from at random.
