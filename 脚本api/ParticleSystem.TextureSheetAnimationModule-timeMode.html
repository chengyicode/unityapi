ParticleSystem.TextureSheetAnimationModule.timeMode

Switch to Manual
public ParticleSystemAnimationTimeMode timeMode;

Description 描述

Select whether the playback is based on mapping a curve to the lifetime of each particle, by using the particle speeds, or if playback simply uses a constant frames per second.
