PlayerSettings.VROculus.lowOverheadMode

Switch to Manual
public static bool lowOverheadMode;

Description 描述

Enable Low Overhead driver optimizations for Oculus.

When enabled, the GLES driver will be initialized with a context that does less validation and error handling. This can result in a performance gain and reduced power usage. If you experience runtime instabilities, disabling this option would be a first step to diagnosing any issues.

This setting currently only applies to mobile GLES players.
