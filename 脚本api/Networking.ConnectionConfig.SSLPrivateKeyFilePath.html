ConnectionConfig.SSLPrivateKeyFilePath

Obsolete public string SSLPrivateKeyFilePath;

Description 描述

Defines the path to the file containing the private key for WebSocket via SSL communication.

SSL WebSocket (WSS) communication is supported. To configure a WSS server, set Networking.ConnectionConfig.SSLCertFilePath and Networking.ConnectionConfig.SSLCAFilePath. If you use self-signed certificates, you don't need to set Networking.ConnectionConfig.SSLCAFilePath.
