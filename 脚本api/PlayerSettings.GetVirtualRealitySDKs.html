public static string[] GetVirtualRealitySDKs(BuildTargetGroup targetGroup);

Parameters

targetGroupThe BuildTargetGroup to return the SDK list for.

Returns

Description 描述

Get the List of Virtual Reality SDKs for a given BuildTargetGroup.
