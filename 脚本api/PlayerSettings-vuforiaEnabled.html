PlayerSettings.vuforiaEnabled

Switch to Manual
public static bool vuforiaEnabled;

Description 描述

Use to enable or disable Vuforia AR for the project.

This value is paired with the Vuforia Augmented Reality setting under the XR Settings tab of the Player Settings. Enabling Vuforia will add the "Vuforia Engine" package to your project and disabling Vuforia will remove the package. This setting cannot be disabled while the Vuforia VR sdk is being used.
