public AndroidETC2FallbackOverride androidETC2FallbackOverride;

Description 描述

ETC2 texture decompression fallback override on Android devices that don't support ETC2.

Allows to choose which texture format to decompress the texture to on Android devices that have no ETC2 texture format support. See Also: AndroidETC2Fallback, AndroidETC2FallbackOverride.
