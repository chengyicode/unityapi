Gizmos.DrawFrustum

public static void DrawFrustum(Vector3 center, float fov, float maxRange, float minRange, float aspect);

Parameters

centerThe apex of the truncated pyramid.
fovVertical field of view (ie, the angle at the apex in degrees).
maxRangeDistance of the frustum's far plane.
minRangeDistance of the frustum's near plane.
aspectWidth/height ratio.

Description 描述

Draw a camera frustum using the currently set Gizmos.matrix for it's location and rotation.
