AnimationCurve.EaseInOut

public static AnimationCurve EaseInOut(float timeStart, float valueStart, float timeEnd, float valueEnd);

Parameters

timeStartThe start time for the ease curve.
valueStartThe start value for the ease curve.
timeEndThe end time for the ease curve.
valueEndThe end value for the ease curve.

Returns

AnimationCurve The ease-in and out curve generated from the specified values.

Description 描述

Creates an ease-in and out curve starting at timeStart, valueStart and ending at timeEnd, valueEnd.
