BatteryStatus

enumeration

Description 描述

Enumeration for SystemInfo.batteryStatus which represents the current status of the device's battery.

Properties 属性

UnknownThe device's battery status cannot be determined. If battery status is not available on your target platform, SystemInfo.batteryStatus will return this value.
ChargingDevice is plugged in and charging.
DischargingDevice is unplugged and discharging.
NotChargingDevice is plugged in, but is not charging.
FullDevice is plugged in and the battery is full.
