TreeView.GetRenameRect

protected Rect GetRenameRect(Rect rowRect, int row, IMGUI.Controls.TreeViewItem item);

Parameters

rowRectRow rect for the item currently being renamed.
rowRow index for the item currently being renamed.
itemTreeViewItem that are currently being renamed.

Returns

Rect The rect where the rename overlay should appear.

Description 描述

Override this method if custom GUI handling are used in RowGUI. This method for controls where the rename overlay appears.

By default the rename overlay matches the placement of the item's displayName text being rendered.
