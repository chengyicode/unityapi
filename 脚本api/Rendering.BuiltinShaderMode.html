BuiltinShaderMode

enumeration

Description 描述

Built-in shader modes used by GraphicsSettings.

See Also: GraphicsSettings.SetShaderMode, BuiltinShaderType, Graphics Settings.

Properties 属性

DisabledDon't use any shader, effectively disabling the functionality.
UseBuiltinUse built-in shader (default).
UseCustomUse custom shader instead of built-in one.
