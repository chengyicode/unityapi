Description 描述

A Struct for the required data for rendering a Tile.

用于渲染Tile所需数据的结构。

Properties 属性

颜色 color Tile的颜色。 Color of the Tile.
标记 flags Tile的TileFlags。 TileFlags of the Tile.
游戏对象 gameObject Tile的GameObject。 GameObject of the Tile.
精灵 sprite 要在Tile上渲染的Sprite。 Sprite to be rendered at the Tile.
变换 transform Tile变换矩阵。 Transform matrix of the Tile.
