BuiltinShaderType

enumeration

Description 描述

Built-in shader types used by GraphicsSettings.

See Also: GraphicsSettings.SetShaderMode, BuiltinShaderMode, Graphics Settings.

Properties 属性

DeferredShadingShader used for deferred shading calculations.
DeferredReflectionsShader used for deferred reflection probes.
LegacyDeferredLightingShader used for legacy deferred lighting calculations.
ScreenSpaceShadowsShader used for screen-space cascaded shadows.
DepthNormalsShader used for depth and normals texture when enabled on a Camera.
MotionVectorsShader used for Motion Vectors when enabled on a Camera.
LightHaloDefault shader used for light halos.
LensFlareDefault shader used for lens flares.
