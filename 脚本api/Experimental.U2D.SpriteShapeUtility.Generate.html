SpriteShapeUtility.Generate

public static int[] Generate(Mesh mesh, Experimental.U2D.SpriteShapeParameters shapeParams, ShapeControlPoint[] points, SpriteShapeMetaData[] metaData, AngleRangeInfo[] angleRange, Sprite[] sprites, Sprite[] corners);

Parameters

meshThe output mesh.
shapeParamsInput parameters for the SpriteShape tessellator.
pointsA list of control points that describes the shape.
metaDataAdditional data about the shape's control point. This is useful during tessellation of the shape.
spritesThe list of Sprites that could be used for the edges.
cornersThe list of Sprites that could be used for the corners.
angleRangeA parameter that determins how to tessellate each of the edge.

Description 描述

Generate a mesh based on input parameters.
