LightmapEditorSettings.filteringGaussRadiusDirect

public static int filteringGaussRadiusDirect;

Description 描述

Specifies the radius the Progressive lightmapper uses to filter for direct light stored in the lightmap when using Gaussian filter. The default value is one. The value range is 0 to 5.

A higher value will increase the strength of the blur, reducing noise from direct light in the lightmap.
